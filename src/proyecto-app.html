Owen Wilson App

Sobre el proyecto

Esta aplicación web fue desarrollada como proyecto final del módulo de React en el bootcamp de Adalab. Se trata de un buscador de series que consume una API externa (TVMaze) para mostrar información en tiempo real. El usuario puede buscar, marcar como favoritas y filtrar resultados de forma dinámica, con almacenamiento en localStorage para mantener el estado. Además, se pueden aplicar filtros por nombre, año de estreno y otros criterios relevantes para afinar la búsqueda. Tecnologías utilizadas: HTML, SCSS, JavaScript y React.

Objetivos

El reto era aplicar todos los conocimientos adquiridos en JavaScript, React, gestión de eventos, estado y props, enrutamiento con React Router, así como persistencia de datos. También se trabajó la accesibilidad, el diseño responsive y el despliegue final de la app.

Resultado

El resultado es una app funcional, ligera y accesible, que permite explorar contenidos televisivos de forma intuitiva. La estructura del proyecto está organizada por componentes, lo que facilita su mantenimiento y escalabilidad futura.
