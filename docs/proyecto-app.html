Owen Wilson App

Sobre el proyecto

Esta aplicación web fue desarrollada como proyecto final del módulo de React en el bootcamp de Adalab. Se trata de un buscador de series que consume una API externa (TVMaze) para mostrar información en tiempo real. El usuario puede buscar, marcar como favoritas y filtrar resultados de forma dinámica, con almacenamiento en localStorage para mantener el estado. Además, se pueden aplicar filtros por nombre, año de estreno y otros criterios relevantes para afinar la búsqueda. Además, cada tarjeta tiene una página detalle donde podrás escuchar los audios de los propios Wow y más información sobre la película. Tecnologías utilizadas: HTML, SCSS, JavaScript y React.

Objetivos

El reto era aplicar todos los conocimientos adquiridos en JavaScript, React, gestión de eventos, estado y props, enrutamiento con React Router, así como persistencia de datos. También se trabajó la accesibilidad, el diseño responsive y el despliegue final de la app.

Resultado

El resultado es una app funcional, ligera y accesible, que permite explorar contenidos televisivos de forma intuitiva. La estructura del proyecto está organizada por componentes, lo que facilita su mantenimiento y escalabilidad futura.

Proyectos Molones App

Sobre el proyecto

“Proyecto Molón” es una aplicación web desarrollada de manera colaborativa. Su objetivo es permitir a cualquier persona crear una tarjeta digital que resuma su proyecto personal de forma visual, clara y profesional. El usuario puede rellenar un formulario con los datos clave (nombre del proyecto, descripción, tecnologías, enlaces…), elegir un estilo y generar una vista previa con un botón para compartir online.

Objetivos

Este proyecto sirvió como práctica de desarrollo colaborativo con React. Aplicamos principios de desarrollo por componentes, uso de formularios controlados, React Router, almacenamiento local y conexión con una API para generar tarjetas compartibles. También pusimos en práctica el trabajo con ramas y control de versiones mediante Git y GitHub, usando issues, boards y pair programming.

Resultado

El resultado es una app funcional, intuitiva y divertida, pensada para facilitar la difusión de proyectos personales de forma sencilla. La interfaz guía al usuario paso a paso en el proceso de creación y ofrece una URL final para compartir su tarjeta personalizada. La experiencia de trabajo en equipo enriqueció no solo el producto final, sino también nuestras habilidades en comunicación (método SCRUM), reparto de tareas y resolución conjunta de problemas.

ProCard Creator App

Sobre el proyecto

Este proyecto fue desarrollado de forma colaborativa en Adalab. Se trata de una aplicación web que permite crear una tarjeta de presentación digital de manera rápida y sencilla. La usuaria puede introducir sus datos, personalizar el diseño y compartir el resultado mediante una URL generada automáticamente.

Objetivos

El objetivo principal del proyecto fue consolidar los conocimientos adquiridos en JavaScript, HTML y SASS, aplicándolos en un desarrollo real en equipo. A lo largo del proyecto trabajamos siguiendo la metodología ágil Scrum, utilizando GitHub como herramienta de colaboración y control de versiones. También practicamos el consumo de APIs externas, el uso de localStorage para guardar datos temporalmente y el diseño responsive para ofrecer una buena experiencia en todo tipo de dispositivos. Además, prestamos especial atención a la accesibilidad y a la organización del código mediante componentes reutilizables y bien estructurados.

Resultado

El resultado es una aplicación intuitiva y funcional, pensada para facilitar la creación de tarjetas digitales profesionales. La interfaz guía a la usuaria paso a paso a través del formulario, la previsualización en tiempo real y la generación de la tarjeta final. El proyecto destaca por su diseño modular, su organización en componentes JS y SCSS, y por el trabajo colaborativo detrás de su desarrollo.
